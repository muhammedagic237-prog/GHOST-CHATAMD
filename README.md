A secure, ephemeral command-line style chat application.

## Features
- **End-to-End Encryption:** Messages are encrypted client-side using AES-256-GCM (Web Crypto API).
- **Ephemeral:** Messages automatically self-destruct after 24 hours.
- **No Logs:** Server only sees encrypted ciphertext.
- **Retro UI:** styled like a CRT terminal.

## Setup
1. Open `index.html`.
2. Enter your **Firebase API Key** and **Project ID** when prompted.
3. Enter a **Username** and **Secret Key**.
4. Share the Secret Key with a friend to chat.
